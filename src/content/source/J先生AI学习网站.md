- 4个阶段
  
    - 1.0 野生 Vibe Coding
      
        - 凭感觉说需求，AI 直接改
          
            - 只能自己用，且效果随机，token风险大
              
    - 2.0 受控 Vibe Coding
      
        - 加规则、加验证
          
            - 可以分享，效果和token可控，可追溯
              
    - 3.0 工程化 Vibe Coding
      
        - 先计划在动手
          
            - (empty)
              
        - 用文件存储上下文
          
        - 切块小步快跑，每步可验证
          
        - 每步有版本管理

        - 可商业化，企业化产品
